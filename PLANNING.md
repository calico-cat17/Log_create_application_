## 기능 목록 (MVP) — 원문 보존
초기 - 로그인 화면으로 아이디와 비밀번호를 입력할 수 있고, 가운데에는 자물쇠 모양 로고가 들어갈 예정이야. 아이디와 비밀번호를 입력하는 칸 하단에는 아이디/비밀번호 찾기와 회원가입 버튼이 있어.

탭2 - 번개 모임 탭에서는 상단에 번개 중/번개 완료 이렇게 2개 탭이 있고, 모집 중인 번개 공고글들은 카드 형식으로 표시된다. 번개 제목, 방탈출/비방탈출, 지역, 날짜, 현재 인원수가 표시되며 신청 시 확인 → 최종 확정 2단계 팝업이 뜬다.

탭3 - 정기모임 탭에서 정모 신청을 누르면 1지망/2지망/3지망 드롭다운 선택창이 나오며, 테마 선택 시 장르·가격이 즉시 표시된다.

탭 4 - 아카이브에는 방탈 랭킹, 후기 모음, 모임 기록이 있으며 각각 더보기로 전체 화면을 볼 수 있다. 모임 기록은 기수별 필터가 가능하며 각 모임을 클릭하면 상세 정보를 볼 수 있다.

탭5 - MY에는 프로필, 스탯 카드, 경고 현황, 설정 목록이 있다. 운영진에게는 부원 관리 메뉴가 추가로 표시된다.

## 프로토타입 UI 수정 사항 (2026-06-21)

### 정기모임 신청 모달
- 테마 선택을 카드 그리드 → 드롭다운 선택창으로 변경
- 1지망 / 2지망 / 3지망 총 3개 지망 지원
- 테마 선택 시 장르·가격 즉시 표시
- 신청 버튼 → "신청을 확정하시겠습니까?" 최종 확인 알림창 추가
- 번개 신청도 동일하게 최종 확인 알림창 추가

### 번개 탭 서브탭 명칭 변경
- "모집 중" → "번개 중"
- "모집 완료" → "번개 완료"

### 운영진 배정하기
- 배정하기 버튼 클릭 → 부원 신청 목록 모달 표시 (1지망/2지망/3지망 + 배정 드롭다운)
- 배정 확정 → "배정이 완료되었습니다." 알림

### 공지 더보기
- 홈 탭 공지의 더보기 → 전체 공지 서브페이지 표시

### MY 탭 기능 확장
- 수정 버튼 → 프로필 수정 화면 (이름, 전화번호, 비밀번호 변경)
- 방탈출 총 횟수 / 선호 장르 / 이번 기수 출석 / 작성한 후기 카드 클릭 → 출석 이력 화면
- 마일리지 카드 클릭 → 마일리지 내역 화면
- 출석 이력 설정 항목 → 출석 이력 화면 (위와 동일)
- 경고 내역 설정 항목 → 경고 내역 조회 화면
- 나의 후기 설정 항목 → 내가 작성한 후기 목록 화면

### 로고 색상
- 좌상단 자물쇠 로고: 분홍/빨강 → 파란색으로 변경

## 프로토타입 UI 수정 사항 (2026-06-21) — 2차

### 공지 작성 (운영진)
- 홈 탭 공지 섹션 헤더에 + 버튼 추가 (운영진만 노출)
- 전체 공지 서브페이지 우측 하단에도 + FAB 버튼 추가
- 공지 작성 화면: 제목 입력칸 + 내용 입력칸 + 취소/업로드 버튼
- 업로드 버튼 → "업로드하시겠습니까?" 최종 확인 알림창 → 확인 시 등록

### 아카이브 — 방탈 랭킹 더보기
- 방탈 랭킹 섹션 헤더에 더보기 버튼 추가
- 클릭 시 전체 랭킹 서브페이지 표시 (1위부터 전체 목록)

### 아카이브 — 모임 기록 더보기
- 모임 기록 섹션 헤더에 더보기 버튼 추가
- 클릭 시 전체 모임 기록 서브페이지 표시
- 기수 별 필터 탭 제공 (전체 / 기수별)

### 아카이브 — 모임 상세
- 모임 기록의 각 항목 클릭 시 상세 화면 표시
- 표시 항목: 제목, 지역, 진행 테마, 참여 인원, 뒷풀이 인원

### 운영진 MY — 부원 관리
- 운영진 로그인 시 MY 탭 설정 목록에 "부원 관리" 항목 추가 (출석 이력 위)
- 부원 관리 서브페이지: 부원 목록 표시
- 각 부원 클릭 → 경고 부여 / 출석 수정 / 마일리지 수정 기능

### 로그아웃 확인
- 로그아웃 버튼 클릭 → "로그아웃하시겠습니까?" 확인 알림창 → 확인 시 로그아웃
