## 기능 목록 (MVP) — 원문 보존
초기 - 로그인 화면으로 아이디와 비밀번호를 입력할 수 있고, 가운데에는 자물쇠 모양 로고가 들어갈 예정이야. 아이디와 비밀번호를 입력하는 칸 하단에는 아이디/비밀번호 찾기와 회원가입 버튼이 있어.

탭2 - 번개 모임 탭에서는 상단에 번개 중/번개 완료 이렇게 2개 탭이 있고, 모집 중인 번개 공고글들은 카드 형식으로 표시된다. 번개 제목, 방탈출/비방탈출, 지역, 날짜, 현재 인원수가 표시되며 신청 시 확인 → 최종 확정 2단계 팝업이 뜬다.

탭3 - 정기모임 탭에서 정모 신청을 누르면 1지망/2지망/3지망 드롭다운 선택창이 나오며, 테마 선택 시 장르·가격이 즉시 표시된다.

탭 4 - 아카이브에는 방탈 랭킹, 후기 모음, 모임 기록이 있으며 각각 더보기로 전체 화면을 볼 수 있다. 모임 기록은 기수별 필터가 가능하며 각 모임을 클릭하면 상세 정보를 볼 수 있다.

탭5 - MY에는 프로필, 스탯 카드, 경고 현황, 설정 목록이 있다. 운영진에게는 부원 관리 메뉴가 추가로 표시된다.

## 프로토타입 UI 수정 사항 (2026-06-23)

### 전체 탭 구조
- 하단 탭을 총 5개로 변경한다: 홈, 모임, 둥지, 후기, 마이.
- 기존 정기모임 탭과 번개 탭을 모임 탭으로 병합한다.
- 기존 아카이브 탭을 후기 탭 중심으로 재구성하고, 둥지 탭을 새로 추가한다.

### 홈
- 최상단에 공지사항 칸을 배치한다.
- 공지사항 칸에는 "확인하지 않은 공지사항이 0건 있습니다" 형식으로 미확인 공지 수를 보여준다.
- 할 일이 있으면 공지사항보다 위에 표시할 카드로 노출한다.
- 그 아래에 다가오는 정기모임과 번개 목록을 기존처럼 요약 카드로 보여준다.

### 모임
- 최상단에 정기 / 번개 구분 세그먼트 컨트롤을 둔다.

#### 정기모임
- 다가오는 모임과 끝난 모임으로 나뉜다.
- 모임 일주일 전보다 이전에는 희망테마 신청하기 버튼을 제공한다.
- 테마 확정 전 상세보기에서는 아직 테마 확정 전이라는 메시지와 희망테마 신청하기 버튼을 보여준다.
- 운영진이 예약한 다음에는 방켓팅 시간 안내 버튼을 비활성 상태로 보여준다.
- 당일에는 방켓팅 버튼과 뒷풀이 버튼을 제공한다.
- 테마 확정 후 상세보기에서는 테마표와 배정표를 보여준다.
- 끝난 모임의 상세 정보, 방켓팅 결과, 테마표, 사진을 확인할 수 있게 한다.
- 운영진 화면에는 배정하기 버튼을 제공하고, 결원이 생기면 같은 버튼으로 재배정할 수 있는 흐름을 보여준다.

#### 정기모임 상세정보
- 제목, 날짜, 위치를 기본으로 보여준다.
- 테마 확정 전에는 확정 전 메시지와 희망테마 신청하기 버튼을 보여준다.
- 테마 확정 후에는 테마표(테마이름, 가격, 모이는 시간, 시작 시간)와 배정표를 보여준다.

#### 번개모임
- 다가오는 번개와 끝난 번개로 나뉜다.
- 다가오는 번개는 번개신청, 모집 인원, 상세정보, 번개올리기 버튼을 제공한다.
- 방탈출 / 비방탈출 구분 필터와 모집마감 안보기 필터를 제공한다.
- 끝난 번개는 내 모임만 보기, 방탈출 / 비방탈출 구분, 상세정보, 사진 확인을 제공한다.

### 둥지
- 코인 잔액으로 기록기를 꾸밀 수 있다.
- 기록기는 알부터 시작해 방수에 따라 성장한다. (방수 = 후기 작성 집계 기준, 출석 횟수와 별개)
- 왼방수 화면에 왼방 표시만큼 성장을 반영한다.
- 코인으로 굿즈, 응모권 등을 구매할 수 있다.
- 기수별로 알 하나씩 부여하며, 해당 기수 기간에만 성장한다.

### 후기
- 방탈랭킹에는 제목, 리뷰 개수, 평균 평점 정보를 보여준다.
- 전체 후기는 별점순, 최신순, 공포도순, 난이도순으로 정렬할 수 있다.

### 마이
- 전체 방탈출 횟수, 기록기 프로필, 이번 기수 출석, 마일리지, 시각화 자료를 보여준다.
- 시각화 자료 영역은 장르별 분포로 한다.
- 세부사항에는 경고 현황, 나의 후기, 로그아웃을 제공한다.
- 운영진 화면에는 부원관리를 제공하며, 부원 출석, 경고, 마일리지 부여 기능을 포함한다.

## 프로토타입 UI 수정 사항 (2026-06-23 2차)

### 전체 구조
- 핸드폰 상태바(시간, 배터리 등)는 프로토타입 화면에서 제외한다.
- 기본 진입 화면을 홈 화면으로 한다.
- 하단 탭을 둥지를 제거하고 홈, 모임, 후기, 마이 4개로 변경한다.
- 둥지의 코인/꾸미기 기능을 마이 탭 하단으로 이동한다.
- 전체 색상에 밝은 흰색 배경, 흰색 카드, 단색 로고/포인트, 선택된 선 기반 보조 포인트를 사용한다.
- 모든 탭에 동일한 _log 로고와 상단 헤더를 적용한다.

### 홈
- 상단에 할 일과 공지를 카드 형태로 표시한다.
- 공지사항이 2개 이상이면 일정 시간마다 아래로 스크롤되며 교체되는 형태로 보여준다.
- 기존처럼 다가오는 정기모임과 번개 요약된 형식을 유지한다.

### 모임
- 정기 / 번개 구분을 탭 형태가 아니라 검은색 글씨 기반의 탭으로 표현한다.
- 정기모임 카드의 상태 배지(오늘 방켓팅, 예약 완료, 테마 확정 전)는 제거한다.
- 운영진 배정하기 버튼을 다가오는 정기모임 카드 오른쪽 위에 배치한다.
- 상세 버튼을 제거하고, 정기모임 제목 또는 카드 클릭 시 상세 표시를 쓴다.
- 예약이 진행되지 않은 모임은 전체화면 상세 대신 스낵 알림창으로 "아직 테마 확정 전입니다."를 보여준다.
- 해당 알림창 하단에는 왼쪽 취소, 오른쪽 희망테마 신청하기 버튼을 배치한다.
- 방켓팅 버튼 클릭 시 1지망부터 3지망까지 방탈출을 선택하는 화면을 보여준다.
- 뒷풀이 버튼 클릭 시 장소, 최대 인원, "뒷풀이에 참여하시겠습니까? 뒷풀이만 참여는 출석으로 인정되지 않습니다." 문구를 보여준다.
- 배정 후의 방켓팅 결과는 표 형태로 출력한다.
- 번개 탭 우측 상단 번개올리기 버튼 및 "다가오는 번개 / 끝난 번개" 문구는 제거한다.

### 후기
- 전체 후기 시 후기 작성 버튼을 제거한다.
- 방탈랭킹과 전체 후기 정렬 필터는 유지한다.

### 마이
- 기존 둥지의 코인/꾸미기 기능을 마이 탭 하단에 추가한다.
- 부원관리 카드의 가로 길이를 나의 후기, 로그아웃과 동일하게 맞춘다.
